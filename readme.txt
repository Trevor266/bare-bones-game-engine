This project is built out as a thin wrapper around linux and windows operating system layers to render out a scalable coordinate space based game.

FOR BUILDING WINDOWS
Run the build.bat file ".\build", "start build", etc. depending on your terminal.

To build the toolkit, which for now is a pretty sparse project that will build into a proper level editor, use 
"buildtoolkit"
